Шпора — 4 царства
🧫
Царство Бактерии
Прокариоты
Нет оформленного ядра — только нуклеоид (кольцевая ДНК).
Нет: митохондрий, пластид, ЭПС, аппарата Гольджи.
Есть: рибосомы, клеточная стенка (не целлюлоза!).
Роли: брожение (йогурт, кефир), гниение, азотфиксация (клубеньки бобовых), болезнетворные.
Споры — для выживания в неблагоприятных условиях, НЕ для размножения.
☣️ RE: T-бактерия Umbrella — прокариот. Антибиотики разрушают клеточную стенку бактерий — именно поэтому они не действуют на вирусы.
🍄
Царство Грибы
Гетеротрофы-эукариоты
Тело = мицелий (грибница) из гиф.
Клеточная стенка — из хитина (как у насекомых).
Запасное вещество — гликоген (как у животных).
Питание: сапрофиты (мёртвая органика) или паразиты.
Сходство с растениями: клеточная стенка, неподвижность.
Сходство с животными: хитин, гликоген, гетеротрофность.
🍄 TLOU: Кордицепс — паразит. Мицелий прорастает сквозь нервную ткань хозяина. Антибиотики против него бессильны — это не бактерия.
🦠
Вирусы + Лишайники
Вне царств живой природы
Вирусы — неклеточная форма жизни.
Состав: нуклеиновая кислота (ДНК ИЛИ РНК) + белковая оболочка (капсид).
Размножаются только внутри клетки-хозяина.
Нет обмена веществ вне клетки — «живой кристалл».
Лишайники — симбиоз гриба + водоросли (или цианобактерии). Биоиндикаторы чистоты воздуха. Не отдельное царство!
☣️ RE: T-вирус Umbrella — РНК-вирус. Антибиотики и антигрибковые препараты против него бесполезны. Нужны противовирусные.
🌿
Растения: эволюция и систематика
«Вросли-Мхи-Папа-Голый-Покрыт»
Вросли = Водоросли → Мхи → Папа = Папоротники →
Голый = Голосеменные → Покрыт = Покрытосеменные
Каждый следующий отдел лучше приспособлен к суше.
Таксоны (от меньшего к большему):
Вид → Род → Семейство → Порядок → Класс → Отдел → Царство (ВИРОСЕ ПОКЛОЦ)
🌻 PvZ: Горох-стрелок, Подсолнух, Вишня-бомба — покрытосеменные, самый эволюционно продвинутый отдел. Семена защищены плодом.

Фотосинтез и ткани растений
6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂
Фотосинтез: CO₂ + H₂O + свет → глюкоза + кислород
Происходит в хлоропластах (фотосинтезирующая паренхима).
Ткани растений:
· Меристема — деление, рост
· Покровная — защита (эпидермис, кора)
· Ксилема — вода вверх ↑
· Флоэма — органика вниз ↓
· Механическая — опора (волокна)
🌻 PvZ: Подсолнух производит «sun» — игровую энергию для всей команды. Прямая аналогия: хлоропласты поглощают свет и синтезируют глюкозу — энергию для клетки.
