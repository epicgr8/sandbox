Квест 3:
Проклятые предметы

Уравнения и неравенства — шпора + 10 испытаний · Baldur's Gate 3

Главные законы
💙
Линейное уравнение
ax + b = 0
Переносишь слагаемые: все с x — влево, числа — вправо.
Затем делишь обе части на коэффициент при x.
⚡ Если ax = 0, то x = 0 (не дели на x, не теряй корень!)
🎮 BG3: Гейл решает уравнение баланса маны. Переносишь лишнее на другую сторону — как выкладываешь лишние артефакты из инвентаря.
💗
Дискриминант
D = b² − 4ac
D > 0 → два корня: x = (−b ± √D) / 2a
D = 0 → один корень: x = −b / 2a
D < 0 → корней нет (в ОГЭ такое тоже встречается!)
⚡ Формула корней: x = (−b ± √D) / 2a
🎮 BG3: D — это индикатор проклятья на предмете. D>0 — два пути снять проклятье. D=0 — один способ. D<0 — предмет нельзя расколдовать.
🧡
Квадратное уравнение
ax² + bx + c = 0
Шаг 1: считаешь дискриминант D = b²−4ac
Шаг 2: если D ≥ 0, считаешь корни по формуле
Шаг 3: проверяешь подстановкой
⚡ Неполное: ax²+c=0 → x²=−c/a → x=±√(−c/a)
🎮 BG3: Проклятый Некромикон требует двух ключей (два корня). Гейл считает дискриминант — есть ли вообще решение, прежде чем тратить заклинание.
💜
Дробное уравнение и ОДЗ
P(x) / Q(x) = 0
ОДЗ — значения x, при которых знаменатель ≠ 0.
Находишь ОДЗ до решения, проверяешь корни после.
Корень, при котором знаменатель = 0 — посторонний, его отбрасываешь.
⚡ Всегда проверяй ОДЗ — это самая частая ловушка ОГЭ!
🎮 BG3: ОДЗ — как список запрещённых заклинаний в Академии Волшебства. Гейл знает: некоторые решения «взорвутся» — их нельзя использовать.
🩵
Линейное неравенство
ax + b > 0
Решается как уравнение, но:
При умножении / делении на отрицательное число — знак неравенства меняется на противоположный!
Ответ записывается как промежуток: x > 3 или x ∈ (3; +∞).
🎮 BG3: Астарион объясняет: «Если умножаешь на отрицательное — всё переворачивается». Как его взгляд на мир после превращения в вампира.
💚
Система неравенств
{ ax > b / cx ≤ d }
Решаешь каждое неравенство отдельно.
Рисуешь числовую ось, отмечаешь оба промежутка.
Ответ — пересечение (где оба выполняются одновременно).
⚡ Если пересечения нет — система не имеет решений.
🎮 BG3: Шэдоухарт и Карлах одновременно указывают безопасный путь. Идти можно только там, где они обе согласны — это пересечение промежутков.
