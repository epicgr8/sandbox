Главные законы
🩷
Сумма углов треугольника
∠A + ∠B + ∠C = 180°
Знаешь два угла — третий считается вычитанием из 180.
🎮 BG3: В партии Гейл, Шэдоухарт, Астарион. Их «уровни доверия» всегда дают ровно 180 в сумме. Один известен — находи остальных.
💙
Теорема Пифагора
a² + b² = c²
Только для прямоугольного! c — гипотенуза (напротив 90°), всегда самая длинная.
🎮 BG3: Гейл телепортируется сначала вдоль стены (катет a), потом поперёк (катет b). Прямой путь — гипотенуза c.
💚
Площадь через высоту
S = ½ · a · h
a — основание, h — высота к нему. Высоту часто надо дорисовать карандашом.
🎮 Tekken: Хитбокс атаки — треугольник. Основание a — ширина удара, h — дальность. Чем длиннее нога Лили — тем больше площадь.
🧡
Площадь через синус
S = ½ · a · b · sin C
Когда даны две стороны и угол между ними.
🎮 Tekken: Казуя бьёт двумя кулаками — левая a, правая b, угол C между ними. Площадь разрушений — это S.
💜
Равнобедренный треугольник
a = b, ∠α = ∠β
Боковые стороны равны. Углы при основании равны. Высота делит основание пополам.
🎮 BG3: Астарион смотрит в зеркало — левая и правая стороны зеркальные. Углы при основании одинаковые, как его улыбка с обеих сторон.
🩵
Равносторонний треугольник
a = b = c, 60° — 60° — 60°
Все три стороны равны. Все три угла по 60°. 180 ÷ 3 = 60.
🎮 Tekken: Идеальная стойка Хворанга — три угла тела одинаковые, баланс абсолютный. Запомни: 60-60-60.
💛
Дорисуй высоту
h ⊥ основание
Задача кажется неразрешимой — проведи высоту. Получишь два прямоугольных треугольника и сразу Пифагор.
🎮 Tekken: Сайдстеп — движение строго перпендикулярно атаке. Высота — это сайдстеп треугольника, всегда под 90°.
❤️
Тройки Пифагора — чит-коды
3-4-5 · 5-12-13 · 8-15-17
Видишь два числа из тройки — третье сразу известно. Считать не надо!
🎮 RPG: Это чит-коды. Выучи как комбо-удар наизусть — экономишь время на экзамене.
