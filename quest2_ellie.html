Квест 2:
Элли против щелкунов

Функции и графики — шпора + 15 испытаний · The Last of Us

Главные законы
💙
Линейная функция
y = kx + b
k — угловой коэффициент (наклон прямой).
b — точка пересечения с осью y.
k > 0 → прямая растёт; k < 0 → убывает.
k = 0 → горизонтальная прямая y = b.
🎮 TLOU: Маршрут Элли — прямая линия. k — это уклон дороги: положительный — подъём, отрицательный — спуск. b — точка старта.
🧡
Парабола
y = ax² + bx + c
a > 0 → ветви вверх (улыбка).
a < 0 → ветви вниз (грусть).
Вершина: x = −b / 2a.
Простейшая: y = x² — вершина в (0;0).
🎮 TLOU: Джоэл бросает кирпич во врага — траектория полёта это парабола. Ветви вниз (a < 0), вершина — наивысшая точка броска.
💜
Гипербола
y = k / x
Две ветви, симметричные относительно начала координат.
k > 0 → ветви в I и III четвертях.
k < 0 → ветви во II и IV четвертях.
Никогда не касается осей (асимптоты x=0, y=0).
🎮 TLOU: Запасы еды и число выживших — чем больше людей, тем меньше еды на каждого. Это гипербола: y = k/x.
💚
Сдвиги графика
y = f(x−a) + b
+b вне скобок → сдвиг вверх на b.
−b вне скобок → сдвиг вниз на b.
f(x−a) → сдвиг вправо на a.
f(x+a) → сдвиг влево на a.
⚠ Сдвиг по x — в обратную сторону от знака!
🎮 TLOU: Карта безопасной зоны сдвинулась — всё то же самое, но на другом месте. График функции тоже «переезжает», не меняя формы.
🔴
Растяжение и отражение
y = a·f(x)
|a| > 1 → растяжение по вертикали.
0 < |a| < 1 → сжатие по вертикали.
a < 0 → отражение относительно оси x.
f(−x) → отражение относительно оси y.
🎮 TLOU: Эбби тренируется — растяжение, как увеличение амплитуды движений. Отражение — зеркальная копия атаки.
🩵
Кусочно-заданная функция
f(x) = { f₁, x<a / f₂, x≥a }
Разные формулы на разных участках области определения.
Строим каждый кусок отдельно на своём отрезке.
Проверяй, включена ли граничная точка (• или ○).
🎮 TLOU: В первой части Элли идёт медленно, после укуса — бежит. Разные «режимы» на разных участках — это и есть кусочная функция.
💛
Свойства функций
↗ возраст. / ↘ убывает
Область определения — все допустимые x.
Область значений — все возможные y.
Нули функции — где y = 0 (пересечение с осью x).
Знак функции — где f(x) > 0 и f(x) < 0.
🎮 TLOU: Томми сканирует местность — он определяет, где безопасно (f > 0), где опасно (f < 0), и где нейтральная зона (нули функции).
