8 признаков живого
КОП · ДАР · ВОР · РОВ
КОП → Клеточное строение + Питание
ДАР → Дыхание + Раздражимость
ВОР → Выделение + Размножение
РОВ → Рост и развитие + Обмен Веществ
Вирусы — не клеточные, поэтому спорны как «живые».
Неживая природа не имеет всех восьми сразу.
🎮 TLOU: Заражённый КЦИ всё ещё живой — КОП ДАР ВОР РОВ. Грибок растёт, дышит, питается. Просто управляет им уже не хозяин.

Методы биологии
Наблюд. / Эксп. / Сравн.
Наблюдение — смотрим без вмешательства.
Эксперимент — меняем условия и смотрим результат.
Сравнительный — ищем сходство и различия.
Исторический — изучаем развитие во времени.
Моделирование — создаём аналог процесса.
В эксперименте: контроль = группа без изменений.
🎮 TLOU: Учёные Светлячков 🕯️ сравнивали иммунитет Элли с клетками обычных людей — сравнительный метод. Тесты на Элли — эксперимент.

Биологические приборы
Микроскоп · Термостат
Световой микроскоп — клетки и ткани (до ×2000).
Лупа — крупные структуры (×2–30).
Термостат — поддерживает температуру в опытах.
Спирометр — измеряет объём лёгких.
Тонометр — измеряет артериальное давление.
рН-метр — кислотность среды.
🎮 TLOU: В лаборатории Светлячков 🕯️ стоит термостат — +37°C, температура тела. Только так КЦИ-культура растёт в пробирке.

Три типа клеток
Раст. / Жив. / Прокар.
Растительная: клеточная стенка (целлюлоза), хлоропласты, крупная вакуоль.
Животная: нет стенки, нет пластид, нет большой вакуоли. Есть лизосомы, клеточный центр. (У высших растений клеточного центра тоже нет, у водорослей — есть.)
Прокариотическая (бактерии): нет оформленного ядра (есть нуклеоид), нет митохондрий, нет ЭПС.
Общее у всех: рибосомы и ДНК.
🎮 TLOU: КЦИ — гриб (эукариот), клетки с ядром. Его не перепутать с бактериями — антибиотики на него не действуют!

1. Молекулярный

4. Органный — орган

6. Организменный — организм
7. Популяционно-видовой

Для ОГЭ главное: Клетка → Ткань → Орган → Система → Организм
🎮 TLOU: КЦИ атакует снизу вверх — сначала клетки мозга, потом ткани, потом весь организм. И в итоге — целая популяция.
🩻
4 типа тканей человека
Эпит. · Соед. · Мыш. · Нерв.
Эпителиальная: клетки плотно прилегают, межклеточного вещества мало. Покрывает тело и выстилает полости.
Соединительная: много межклеточного вещества. Включает кровь, кость, хрящ, жир.
Мышечная: сократимость. Гладкая (непроизвольная), поперечнополосатая (произвольная), сердечная.
Нервная: нейроны (отростки) + нейроглия. Возбудимость и проводимость.
🎮 TLOU: КЦИ прорастает сквозь нервную ткань мозга — нейроны имеют длинные отростки, и по ним грибок распространяется быстрее всего.
