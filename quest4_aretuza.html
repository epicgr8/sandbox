Алгебра
Формулы сокращённого умножения
(a+b)² = a² + 2ab + b² (a−b)² = a² − 2ab + b² (a+b)(a−b) = a² − b²
⚠ Главная ловушка: (a+b)² ≠ a²+b² — нельзя забывать 2ab!
Алгебра
Дискриминант и корни
D = b² − 4ac x = (−b ± √D) / 2a D > 0 → два корня D = 0 → один корень D < 0 → корней нет
⚠ В числителе −b, не b. И делить нужно на 2a целиком!
Корни
Свойства корней (знать наизусть!)
√(a·b) = √a · √b √(a/b) = √a / √b √(a²) = |a| a^(1/n) = ⁿ√a
⚠ √(a²) = |a|, не просто a! При a<0 это важно.

△: S = ½·a·h □: S = a·b ◇ ромб: S = ½·d₁·d₂ ○ круг: S = π·r² ▱ паралл.: S = a·h △ через sin: S = ½·a·b·sinC
Тригонометрия
Основные соотношения (прямоугольный △)
sin α = противолежащий / гипотенуза cos α = прилежащий / гипотенуза tg α = sin α / cos α sin²α + cos²α = 1
Мнемоника: СОЧ ПОЧ — Синус Отношение Противолежащего, Косинус Отношение Прилежащего.
Окружность
Вписанные и центральные углы
Центральный = дуге Вписанный = ½ · дуги Вписанный на полуокружности = 90° Длина окружности: C = 2πr Внешний угол △ = сумма 2 несмежных
Вероятность
Классическая вероятность
P(A) = m / n m — число благоприятных исходов n — число всех равновозможных 0 ≤ P(A) ≤ 1
Вероятность никогда не бывает больше 1 или меньше 0.
